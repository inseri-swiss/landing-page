inseri vision

inseri connects data providers, web components, local user data and microservices, so that a user can combine them in any possible way.
It serves as an online collaborative work and publishing environment.

↑ ↓

web components

inseri can integrate any own or third party web component as an inseri app.
Any inseri app can be used with own user data, existing data providers and microservices


↑ More ↓

Data Providers
